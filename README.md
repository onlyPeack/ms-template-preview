# avue-cli

这是基于vue-cli@3.0改的一个脚手架  

## Run vueUI
* 1.npm install -g @vue/cli  全局安装vue脚手架最新版
* 2.vue --version 查看版本是否为3.x版本
* 3.vue-ui 运行管理工具，导入avue-cli项目
* 4.剩下的自己摸索吧，下面时图示

[参考博客](https://segmentfault.com/a/1190000015366009)

## Run setup
```
cd avue-cli
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
